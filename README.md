# Marketing Campaign Performance Analysis – Power BI

## Objective
Analyze Meta marketing campaign data to evaluate spend efficiency, lead generation, and revenue impact across cities, demographics, and time.

## Tools Used
- Power BI
- DAX
- SQL (data preparation)
- Excel / CSV

## Key KPIs
- Total Spend
- CTR
- Leads (Estimated – 5% benchmark)
- ROAS
- CPL
- ROI %

## Business Impact
- Identified high-spend, low-conversion campaigns
- Highlighted funnel drop-offs from impressions to leads
- Enabled data-driven budget reallocation decisions

## Notes
Leads are estimated using a 5% click-to-lead benchmark due to absence of CRM integration.
